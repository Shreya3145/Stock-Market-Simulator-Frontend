# Stock Trading App (Frontend Skeleton)

A Flutter-based stock trading application skeleton that defines the core structure and navigation for a trading platform.
This project focuses on UI architecture, screen flow, and modular design rather than full feature implementation.

---

## Tech Stack

* Flutter
* Dart

---

## Features

* Basic app structure with multiple screens
* Navigation between core modules
* Modular UI design for scalability

---

## Modules

### 1. Homepage

* Entry point of the application
* Provides navigation to login and registration

### 2. Authentication

* Login screen
* Registration screen
* Basic form structure (no backend integration yet)

### 3. Dashboard

* Central hub after login
* Placeholder for portfolio summary and quick actions

### 4. Stock

* Displays stock-related UI structure
* Placeholder for stock listing and details

### 5. Trade

* UI skeleton for buy/sell functionality
* Input fields and layout for future integration

---

## Current Status

* UI skeleton completed
* Navigation flow implemented
* Backend integration not implemented
* Business logic is minimal
